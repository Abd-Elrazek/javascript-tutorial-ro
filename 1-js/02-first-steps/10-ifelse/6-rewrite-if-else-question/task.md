# Rescrie 'if..else' în '?'

Rescrie `if..else` folosind operatori ternari `'?'` multiplii.

Pentru lizibilitate este recomandat să divizezi codul în linii multiple.

```js
let message;

if (login == 'Employee') {
  message = 'Hello';
} else if (login == 'Director') {
  message = 'Greetings';
} else if (login == '') {
  message = 'No login';
} else {
  message = '';
}
```
